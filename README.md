>**1x** LCD Screen
>**1x** Switch M/A

>**1x** green arcade button
>**4x** black arcade button
>**1x** red arcade button
>**1x** Arduino méga 2560
>   Jumpers
>**1x** Arduino speaker
>**1x** 12V 3A Power supply
>**20x** 10kohm resistance
